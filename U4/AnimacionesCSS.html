Animacines CSS

Las animaciones y transiciones en CSS son herramientas potentes que permiten añadir movimiento e interacción visual en las páginas web, mejorando la experiencia del usuario. Con estas técnicas, puedes controlar como los elementos cambian sus propiedades (como color, posición u opacidad) de manera suave y fluida.

  1. Uso de transiciones:

    Las transiciones CSS permiten que los cambios en las propiedades de un elemento, como el color, la opacidad, la medida o la posición, sean suaves en lugar de bruscos. Cuando una propiedad cambia, puedes definir la velocidad del cambio utilizando transition. 

            :root {
                --color-primari: #3498db;
                --marge-petit: 10px;
            }
            body {
                background-color: var(--color-primari);
                margin: var(--marge-petit);
            }
