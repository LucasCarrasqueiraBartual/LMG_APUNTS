Introducción

Los selectores avanzados en CSS permiten aplicar estilos con mucha más precisión y flexibilidad, permitiendo seleccionar elementos específicos en función de su posición, relación con otros elementos o atributos.

Selector Avanzados

  1. Selectores de atributo

    Los selectores de atributo permiten aplicar estilos a los elementos HTML que contienen un determinado atributo o valor de atributo. Son especialmente útiles para estilizar elementos de formularios o en casos en que utilices atributos personalizados. 

                            input[type="text"] {
                            border: 1px
                        }

                        a[href*="example"] {
                            color: green;
                        }
  2. Selectores Combinados

    Los selectores combinados permiten seleccionar elementos que tienen una relación específica con otros elementos dentro del documento. Los combinadors más comunes son: 

                            div p {
                            color: red;
                        }

                        div > p {
                            color: blue;
                        }

                        h1 + p {
                            font-weight:
                        }

                        h2 ~ p {
                            color: gray;
                        }
  3. Selectores pseudo-classe

    Las pseudo-clases son selectores que se utilicen para aplicar estilos a un elemento en un estado o condición particular. Esto permite cambiar el estilo de un elemento cuando el usuario interactúa o cuando el elemento cumple ciertos criterios.

    1. :hover - Aplica estilos cuando el usuario pasa el ratón por encima del elemento.
    2. :nth-child() - Selecciona elementos en función de su posición entre sus hermanos.
    3. :first-of-type: Selecciona el primer elemento de su tipo dentro de su padre.
    4. :not(): Selecciona todos los elementos excepto aquellos que coincidan con el selector especificado. 
                            a:hover {
                            color: red;
                        }

                        tr:nth-child(odd) {
                            background-color: #f0f0f0;
                        }
                        
                        p:first-of-type {
                            font-weight: bold;
                        }

                        div:not(.important) {
                            background-color: yellow;
                        }
  4. Selectores pseudo-elemento

    Los pseudo-elementos permiten aplicar estilos a una parte específica de un elemento, como su contenido antes o después, o una línea de texto. Los pseudo- elementos más comunes son ::before y ::after. 

                            h1::before {
                            content: "★ ";
                            color: gold;
                        }

                        p::after {
                            content: " (Final)";
                            color: gray;
                        }
