Etiquetas HTML

<center>
Se utilizaba para centrar contenido pero en HTML5 esta obsoleta

<div>
Se utiliza para crear contenedores de blocs en una pagina HTML.
No tiene significado semantico y suele ser estilada con CSS
Util para organizar y diseñar diferentes secciones

<span>
Es una etiqueta inline que sirve para estilizar o manipular pequeñas porciones
No es semantica y se usa principalmente para aplicar estilos

<hr>
Se usa no solo decorativo sino semantico para indicar un cambio de tema o ruptura

<template>
Se utiliza para definir fragmentos HTML reutilizables que no se muestren inmediatamente cuando cargue.
El contenido no se muestra hasta que se manipula explicitamente con JavaScript

<Iframes>
Es una etiqueta que permite insertar una pagina HTML en otra web.
Es una manera segura de incrustar contenido externo, se suele utilizar para videos de YouTube, Google Maps...

<code>
Se utiliza para marcar un texto con formato codigo y darle su propia fuente para representacion

<pre>
Se usa para formatear un texto de la misma manera de la que se ha pegado del portapapeles
