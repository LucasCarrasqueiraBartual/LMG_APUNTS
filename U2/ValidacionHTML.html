Validacion de Formularios

HTML5 introduce una serie de atributos y tipos de entrada para validar los formularios
de manera nativa, sin usar JavaScript.

Atributo requerido

  • Marca un campo como obligatorio
    <input type="text" name="nom" required placeholder="Escriu el teu nom">

Atributo pattern

  • Permite definir un patron con expresiones regulares para validar la entrada del usuario.
    <input type="text" name="telefon" pattern="\d{9}" placeholder="Escriu el teu tel√®fon">
