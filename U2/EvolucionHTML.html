HTML5 y su evolución

Es una evolución importante respecto a las versiones anteriores de HTML,
introduciendo nuevas etiquetas y funcionalidades que permiten una mejor gestión
del contenido multimedia, formularios avanzados y mejoras en la semántica de la página web.

Etiquetas en HTML5

  1. <section>
    Se usa para agrupar contenido relacionado en una pagina.

  2. <article>
    Representa un bloque de contenido autonomo, como un aritculo de blog
    una noticia o un post.

  3. <aside>
    Define un contenido relacionado con el contenido principal pero no esencialmente.

  4. <header> y <footer>
    Etiquetas para definir los encabezados y pie de pagina

  5. <nav>
    Especifica para las barras

  6. <figure> y <figcaption>
    Agrupa elementos multimedia y sus descripciones
IMPORTANTE
Buenas Prácticas para Utilizar Elementos de estructura
  • Utiliza elementos semánticos para organizar mejor la página.
    Esto hace que sea más comprensible para los navegadores, buscadores y lectores de pantalla.

  • Evita el uso excesivo de divs ("divitis") y aprovecha etiquetas como <section>, <article> y <aside> para hacer que la página sea más semántica.

  • Proporciona enlaces de navegación claros dentro de los elementos <header> y <footer> para mejorar la experiencia de usuario y la accesibilidad.

  • No abuses de <aside>, ya que este elemento debe contener información complementaria y no esencial para el contenido principal.
