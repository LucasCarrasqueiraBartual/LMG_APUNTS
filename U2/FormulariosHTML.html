Formularios con HTML

Los formularios son una de las herramientas más importantes de HTML porque permiten a los usuarios enviar datos a un servidor.
Con HTML5, los formularios han ganado nuevas funcionalidades que mejoran su usabilidad y validación.

Un formulario se crea con la etiqueta <form>, que actúa como el contenedor de todos los elementos de entrada (texto, botones, selecciones, etc.).

Ejemplo 

        <form action="submit.php" method="POST">
        <!-- Aquí van los campos del formulario -->
        </form>

Etiquetas

  • <input> Es el elemento mas versatil para crear campos de entrada. El tipo de entrada se define usando type
  • <label> Se utiliza para etiquetar campos de formularios
  • <textarea> Para textos largos se utiliza <textarea> que permite area de texto mas grande
  • <select> Se utiliza para crear listas desplegables de selección. Cada opcion se define con <option>
    Con posibilidad de añadir un atributo multiples opciones en una lista.
  • <button> Se utiliza para crear botones de enviamiento o otras acciones.
  • radio buttons Permiten seleccionar una opcion de un conjunto de posibilidades
  • checkboxes Permiten seleccionar diversas opciones independientemente

Ejemplos de uso (por orden de explicacion)

<input type="text" name="nom" placeholder="Escriu el teu nom">
<label for="nom">Nom:</label>
<input type="text" id="nom" name="nom">
<textarea name="comentari" rows="4" cols="50" placeholder="Escriu el teu comentari aquí"></textarea>

<select id="ciutat" name="ciutat">
<option value="barcelona">Barcelona</option>
<option value="madrid">Madrid</option>
<option value="valencia">València</option>
</select>

<button type="submit">Enviar</button>
<label><input type="radio" name="sexe" value="home"> Home</label>
<label><input type="checkbox" name="subscripcio" value="noticies"> Subscriu-me a les notícies</label>
