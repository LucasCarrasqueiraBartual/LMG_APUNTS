Introducción HTML

HTML (HyperText Markup Language) es el lenguaje de marcas para crear páginas web.
Define la estructura de la página con etiquetas que el navegador interpreta para mostrar contenido.

Breve Historia

  1. Creado por Tim Berners-Lee el 1991.
  2. HTML 2.0 (1995): Primer estándar oficial.
  3. HTML5 (2014): Última versión con nuevas funcionalidades como apoyo para video, audio y gráficos.

Estructura básica

Un documento HTML se basa en una estructura jerárquica de etiquetas que definen las partes de una página.
Cada página HTML sigue una estructura similar, empezando por el <!DOCTYPE>, seguido de las etiquetas <html> <head> <body>.

Etiquetas básicas

<title>

 Define el titulo que aparece en la pestaña del navegador o a los resultados de los buscadores.

<h1> a <h6>

 Utilizado para definir titulos y subtitulos a la página web.

<p>

 Se utiliza para definir bloques de texto/paragrafos.

<br>

 Fuerza un texto a comenzar en otra linea.

& nbsp;

 Inserta un espacio en blanco que no se rompe entre lineas


Ejemplo de uso 
            <!DOCTYPE html>
            <html>
            <head>
                  <title>Hello world</title>
            </head>
            <body>
                <h1>titulo 1</h1>
                <h2>titulo 2</h2>
                <h3>titulo 3</h3>
                <h4>titulo 4</h4>
                <h5>titulo 5</h5>
                <h6>titulo 6</h6>
                <p>Esto es un parágrafo con información.</p>
                <p>Este texto se dividira<br>en dos lineas.</p>
                <p>Esta es la frase "12&nbsp;hombres".</p>
            </body>
            </html>
