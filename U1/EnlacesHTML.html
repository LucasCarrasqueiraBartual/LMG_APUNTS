Enlaces en HTML

  • La etiqueta <a> se utiliza para crear enlaces a otras páginas web, documentos, secciones dentro de la página...

  • El elemento esencial para los enlaces es el atributo href, que define la URL a la que redirige al usuario.

  • El texto entre la etiqueta de apertura y la de cierre es lo que los usuarios verán y podrán hacer clic para ser redirigidos.

Atributos de Enlaces

  1. href (hypertext reference)
    Definie la URL de la destinación del enlace. Puede ser relativo o absoluta.

  2. target
    Define donde se abrira el enlace (misma pestaña, una nueva...)
    Los valores habituales son:
    • _blank: Abre el enlace en una nueva pestaña
    • _self: Abre el enlace en la misma pestaña (predeterminado)

Ejemplo de Enlace

<a href="https://www.google.com">Cerca a Google</a>
IMPORTANTE
Buenas Prácticas con Enlaces
  • Utiliza enlaces descriptivos en lugar de genéricos como "Haz clic aquí".
    Esto mejora la accesibilidad y el SEO.

  • Utiliza target="_blank" solo cuando sea realmente necesario,
    ya que abrir enlaces en nuevas pestañas puede confundir a algunos usuarios.

  • Siempre asegúrate de que los enlaces sean accesibles añadiendo una descripción adecuada
    para usuarios con lectores de pantalla.
