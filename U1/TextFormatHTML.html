Texto con Formato

<b> y <strong>: Negrita

<b> aplica negrita sin indicar importancia semántica.
<strong> aplica negrita indicando que el texto es importante.

<i> y <em>: Cursiva

<i> aplica cursiva sin implicar enfasis.
<em> aplica cursiva con enfasis indicando que el texto es importante

<u>: Subrayado

Se utiliza para subrayar texto.

<strike> o <s>: Texto rayado

Aplica un rayado al texto, indicando que esta borrado o no es relevante.

<sup>: Superíndex

Se utiliza para elevar el texto, como en exponentes matematicos.

<sub>: Subindex

Se utiliza para poner texto en la posición de subindex.
