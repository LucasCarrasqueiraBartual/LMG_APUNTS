Listas HTML

Tipo de Listas

  • Listas no ordenadas <ul>
    1. Se utilizan para listas de elementos sin un orden específico.
    2. Els elements es mostren amb vinyetes o altres símbols abans de cada ítem.
    3. Cada elemento de la lista se define con la etiqueta <li> (list item).
  • Listas Ordenadas <ol>
    1. Se utilizan para listas en las que el orden de los elementos es relevante.
    2. Los elementos se numeran automáticamente (1, 2, 3, etc.).
    3. Cada elemento de la lista también se define con <li>.

Atributos de listas

  • Atribut type (para listas ordenadas)
    1. Se puede utilizar para definir el tipo de numeración en la lista
      1: Nombres
      A: Mayusculas
      a: Minusculas
      I: Numeros romanos en Mayusculas
      i: Numeros romandos en Minusculas
  • Atribut start (para listas ordenadas <ol>)
    • Define el numero o letra para comenzar la numeración

Ejemplo de Listas

        <ol type="A">
            <li>Primer ítem</li>
            <li>Segon ítem</li>
        </ol>
        <ol start="5">
            <li>Ítem número 5</li>
            <li>Ítem número 6</li>
        </ol>
IMPORTANTE
Buenas Prácticas con Listas
  • Utiliza listas no ordenadas (<ul>) cuando el orden de los ítems no sea importante.

  • Utiliza listas ordenadas (<ol>) cuando el orden sea relevante.

  • Personaliza las listas según las necesidades visuales o estilísticas
    utilizando los atributos adecuados.
