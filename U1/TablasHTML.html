Tablas basicas HTML

Etiquetas

  • Etiqueta <table>: La etiqueta <table> define la tabla y actua como contenedor principal
  • Etiqueta <tr>(table row): Define una fila dentro de la tabla.
  • Etiqueta <td> (table data): Crea una celda dentro de una fila de una tabla.

Atributos de Tabla

  • border: Añade un borde de tabla y a las celdas
  • cellpadding: Define el espacio entre el contenido de las celdas y de los bordes de celda.
  • cellspacing: Controla el espacio entre celdas de la tabla.

Tablas en HTML

  • Etiqueta <thead>
    Define el encabezado de la tabla, agrupando las filas de encabezado.

  • Etiqueta <tbody>
    Agrupa el cupero de la tabla, las filas de datos.

  • Etiqueta <th>
    Se utiliza para crear una celda de encabezado en una tabla. El texto dentro de <th>
    aparece en negrita y centrado por defecto.
IMPORTANTE
Buenas Prácticas con Tablas
  • Utiliza <thead> y <tbody> para organizar las tablas, especialmente en tablas más grandes,
    para separar claramente el encabezado del cuerpo de la tabla.

  • Mantén las tablas simples para datos tabulados y utiliza CSS para el diseño más avanzado de las tablas
    (colores, tamaño de la fuente, etc.).
