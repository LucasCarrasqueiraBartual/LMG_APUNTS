Elementos HTML y anidamiento

Diferencia entre Etiquetas y Elementos

Etiqueta (Tag)
  • Es la parte que define el inicio y final
    de un componente en HTML.

  • Se delimita con los simbolos < >
    Por ejemplo: <p>, </p>
Elemento
  • Un elemento incluye la etiqueta de apertura,
    el contenido (si hay), y la etiqueta de cierre.

  • Los elementos pueden estar vacios
    o contener otros elementos dentro.
    Por ejemplo: <p>Esto es un parrafo</p>

Elementos Anidados

  • Se refiere a colocar un elemento HTML dentro de otro.

  • Es importante seguir un orden de las etiquetas.
