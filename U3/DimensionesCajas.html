Introducción

El CSS utiliza el modelo de caja para representar los elementos de una página web. Cada elemento se considera como una caja rectangular que contiene el contenido, rellenados, márgenes y bordes. Es importante entender este modelo porque controla como se distribuyen y se ajustan los elementos dentro de la página.

El modelo de caja se compone de cuatro partes principales:

  1. Contenido: El contenido real del elemento, como texto, imágenes u otros elementos.
  2. Relleno (padding): El espacio entre el contenido y el borde.
  3. Borde (border): El límite alrededor del relleno.
  4. Margen (margin): El espacio externo entre el borde y otros elementos.

Dimensiones y Cajas

  1. Propiedad de dimensiones (width y height)

    Las propiedades width (anchura) y height (altura) se utilizan para definir la medida de un elemento. Estos valores se pueden definir en unidades fijas como píxeles (px) o unidades relativas como porcentajes (%), que dependen de la medida de su elemento contenedor. 

                        div {
                        width: 300px;
                        height: 150px; 
                    }
                    .container {
                        width: 50%;
                        height: auto; 
                    }
  2. El modelo de caja en CSS

    El modelo de caja establece como se calculan las dimensiones de un elemento teniendo en cuenta las propiedades de márgenes, rellenados, y bordes. Por defecto, la medida de width y height solo afecta el contenido. Para incluir el relleno y el borde dentro de las dimensiones del elemento, se puede utilizar la propiedad box-sizing. 

                        div {
                        width: 200px;
                        padding: 20px;
                        border: 5px solid black;
                        box-sizing:
                    }
  3. Control del desbordamiento (Overflow)

    Cuando el contenido de un elemento es más grande que las dimensiones especificadas, lo CSS ofrece la propiedad overflow para controlar como se muestra el contenido que desborda
    Los valores mas comunes son:

    • visible El contenido que desborda se muestra fuera del elemento (valor por defecto).
    • hidden El contenido que desborda se recorta y no se muestra.
    • scroll Muestra barras de desplazamiento, aunque el contenido no desborde.
    • auto Muestra barras de desplazamiento solo si el contenido desborda. 
                        div {
                        width: 150px;
                        height: 100px;
                        overflow: scroll;
                    }
