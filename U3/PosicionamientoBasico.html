Introducción

Lo CSS ofrece varias técnicas para controlar la ubicación de los elementos en una página web. La propiedad clave para el posicionamiento es position, que tiene varios valores para definir como se posiciona un elemento respecto a su elemento contenedor o al documento.

Posicionamiento Basico

  1. Propiedad de position
    1. static (por defecto): Los elementos se posicionan de manera normal, siguiendo el flujo del documento. No se pueden ajustar manualmente con coordenadas.
    2. relative: El elemento se posiciona respecto a su posición original en el flujo del documento. Se pueden utilizar coordenadas para moverlo ligeramente (arriba, abajo, izquierda o derecha).
    3. absolute: El elemento se saca del flujo del documento y se posiciona respecto a su elemento contenedor más cercano que tenga una posición definida. Si no hay ningún contenedor definido, se posiciona respecto a la página.
    4. fixed: El elemento se saca del flujo del documento y se posiciona respecto a la ventana del navegador. Se mantiene fijo incluso cuando el usuario se desplaza.
    5. sticky: El elemento se comporta como relative hasta que llega a un punto en el desplazamiento de la página, después del cual se comporta como fixed.
  2. Propiedades de posicionamiento

    Cuando se define la propiedad position como relative, absolute, fixed o sticky, puedes utilizar las propiedades top, right, bottom, y left para especificar la distancia del elemento respecto a sus límites. 

                        div {
                        position: relative;
                        top: 20px; /* Mou l'element 20 píxels cap avall des de la seva posició original */
                        left: 30px; /* Mou l'element 30 píxels cap a la dreta */
                    }
  3. Posicionamiento absoluto

    Con position: absolute, el elemento se saca completamente del flujo normal del documento y se posiciona respecto a su elemento contenedor más próximo con position: relative, absolute, o fixed. 

                        div {
                        position: absolute;
                        top: 50px;
                        left: 100px;
                    }
  4. Posicionamiento fix

    Los elementos con position: fixed se mantienen en una ubicación fija a la pantalla incluso cuando el usuario hace desplazamientos. Se posicionan respecto a la ventana del navegador. 

                        div {
                        position: fixed;
                        top: 0;
                        left: 0;
                        width: 100px;
                        height: 100px;
                        background-color: red;
                    }
  5. Posicionamiento sticky

    Los elementos con position: sticky se comportan como relative hasta que el usuario se desplaza por la página y llega a un punto especificado, después del cual el elemento se comporta como fixed. 

                        h1 {
                        position: sticky;
                        top: 10px;
                    }
  6. Propiedad z-index

    La propiedad z-index se usa para controlar la orden de superposición de los elementos posicionados (relative, absolute, fixed o sticky). Los valores más altos de z-index colocan el elemento ante otros elementos con valores inferiores. 

                        div {
                        position: absolute;
                        z-index: 10;
                    }
