Introducción

Las Media Queries son una característica poderosa del CSS que permite adaptar el diseño de una página web en función de las características del dispositivo que lo está visualizando, como la medida de la pantalla, la resolución o la orientación. Esta técnica es clave para implementar diseños responsius que ofrecen una experiencia óptima tanto en dispositivos móviles como en tabletas u ordenadores de sobremesa. 

            @media (condició) {
                /* Estils que només s'aplicaran si es compleix la condició */
            }

Media Queries Introduccion

  1. Aplicacion basica de media queries para pantallas de distinta medida

    Un uso típico de las media queries es adaptar el diseño a diferentes anchuras de pantalla. Por ejemplo, puedes ajustar los estilos para pantallas grandes y pequeñas. 

                        @media (min-width: 768px) {
                        body {
                            background-color: lightblue;
                        }
                    }
                        
                    @media (max-width: 767px) {
                        body {
                            background-color: lightgreen;
                        }
                    }
  2. Tipo de condiciones mas comunes

    A continuación, veremos algunas de las condiciones más comunes que puedes utilizar dentro de las media queries:

    1. Anchura de la pantalla (min-width y max-width): Estas condiciones se utilizan para aplicar estilos cuando la ventana del navegador tiene una anchura mínima (min-width) o máxima (max-width).
    2. Altura de la pantalla (min-height y max-height): Estas condiciones funcionan de manera similar a las de anchura, pero se basan en la altura de la ventana.
    3. Orientación de la pantalla (orientation): Puedes utilizar la condición orientation para aplicar estilos diferentes si la pantalla está en orientación horizontal (landscape) o vertical (portrait).
                        @media (orientation: landscape) {
                        /* Estils per a dispositius en orientació horitzontal */
                    }
                    @media (orientation: portrait) {

                    }
  3. Uso de Multiples condiciones de query

    Puedes combinar varias condiciones en una sola media query mediante las palabras clave and o only para especificar estilos muy concretos. 

                        @media (min-width: 600px) and (max-width: 1200px) {
                    
                    }
