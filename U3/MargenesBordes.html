Introducción

Los márgenes (margin), los rellenos (padding) y los bordes (border) son parte del modelo de caja en CSS, que describe como se disponen los elementos en la página. Estas propiedades permiten ajustar el espacio exterior e interior de un elemento, así como definir los límites visuales con un borde.

Margenes, rellenos y los bordes

  1. Margenes (margin)

    La propiedad margin controla el espacio exterior de un elemento, es decir , el espacio que hay entre un elemento y los otros elementos que lo rodean. Los márgenes se pueden definir de manera individual para cada uno de los cuatro lados del elemento (superior, derecho, inferior, izquierdo) o de manera global para todos los lados a la vez. 

                      p {
                    margin: 10px; 
                  }
                  h1 {
                    margin-top: 20px; 
                    margin-bottom: 10px; 
                    margin-left: 10px; 
                    margin-right: 10px; 
                  }
  2. Rellenos (padding)

    La propiedad padding controla el espacio interior de un elemento, es decir , el espacio entre el contenido de un elemento (como texto o imágenes) y sus bordes. Igual que con los márgenes, los rellenos se pueden definir para todos los lados o para cada uno individualmente. 

                      div {
                    padding: 20px; 
                  }
                  h2 {
                    padding-left:
                  }
  3. Bordes (border)

    La propiedad border se utiliza para añadir un borde alrededor de un elemento. Esta propiedad tiene diversos subatributos:

    • Anchura del borde (border-*width): define el grosor del borde.
    • Estilo del borde (border-*style): puede ser solid (sólido), dashed (rayado), dotted (puntitos), etc.
    • Color del borde (border-color): especifica el color del borde. 
                      div {
                    border: 2px solid black; 
                  }
                  p {
                    border: 1px dashed blue; 
                  }

Unidades Comunes

  1. Píxeles (px): Unidad fija, representa un punto a la pantalla. Ejemplo: margin: 20px;
  2. Ems (me): Unidad relativa a la medida de la fuente del elemento pare. Ejemplo: padding: 2em; (2 veces la medida de la fuente).
  3. Remos (remo): Unidad relativa a la medida de la fuente del elemento raíz (HTML). Ejemplo: margin: 1.5rem; (1.5 veces la medida de la fuente raíz).
  4. Porcentaje (%): Relativa a la anchura o altura del contenedor pare. Ejemplo: padding: 5%; (5% de la anchura del contenedor).
  5. Viewports (vw, vh): Relativa a la medida de la ventana visible. Ejemplo: margin: 5vw; (5% de la anchura del viewport).
  6. Bordes (border-*width): Grosores predeterminados: thin, medium, thick
