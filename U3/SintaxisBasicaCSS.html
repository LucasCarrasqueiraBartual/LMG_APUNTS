Sintaxis Basica de CSS

  1. Regla de CSS: Es una instrucción que define como se tiene que aplicar un estilo a uno o más elementos HTML dentro de una página web.

  2. Selector: Indica el elemento HTML al cual se aplicarán los estilos.

  3. Propiedad: Defineix la característica que vols modificar (com ara el color, la mida del text, el marge, etc.).

  4. Valor: Asigna un valor específico a la propiedad (como por ejemplo "moratón" para el color, "16px" para la medida del texto, etc.).

  5. Declaracion CSS: Es una instrucción dentro de una regla CSS que define una característica específica de estilo para un elemento HTML.

Selectores en CSS

Los selectores son una parte fundamental del CSS porque definen qué elementos HTML serán afectados por las reglas de estilo. Hay varios tipos de selectores, pero los más comunes en el aprendizaje básico son:

  1. Selector de tipo (Type Selector):

    Este selector aplica estilos a todos los elementos HTML del mismo tipo. Este código aplicará el color verde en todos los títulos (<h1>) de la página. 

                        h1 {
                        color: green;
                    }

  2. Selector de clase (Class Selector):

    Los selectores de clase permiten aplicar estilos a elementos HTML con una clase específica. Se utiliza un punto (.) para seleccionar una clase. Cualquier elemento HTML que tenga el atributo class="mí- clase" tendrá un fondo amarillo. Por ejemplo: 

                        .mi-classe {
                        background-color: yellow;
                    }

  3. Selector de id (ID Selector):

    El selector de id se utiliza para aplicar estilos a un único elemento HTML que tenga un identificador específico. Se usa el símbolo de la coixinera #. Este ejemplo centrará el texto del elemento que tenga id="mí-*id". Es importante recordar que un ID tiene que ser único en la página. 

                        #mi-id {
                        text-align: center;
                    }

  4. Selector universal:

    El selector universal (*) aplica estilos a todos los elementos de una página. Este ejemplo elimina los márgenes y rellenados de todos los elementos de la página, cosa útil para establecer un estilo uniforme. 

                        * {
                        margin: 0;
                        padding: 0;
                    }

Conceptos Importantes

  • Herencia: Los hijos heredan los estilos de sus padres, así no hace falta que escribas estilos para todo. 
    •                     body {
                        color: blue;
                    }
                    h1{
                        color:blue;
                    }
  • Especificado: Cuando hay conflictos de estilos, el navegador aplica solo el de más especificidad. 
    •                     .parrafo {
                        color: blue;
                    }
                    .article .parrafo{
                        color:red;
                    }
  • Cascada: Todo estilo sobrescribe uno de anterior. La C de CSS significa Cascada. 
    •                     .parrafo {
                        color: blue;
                    }
                    .parrafo{
                        color:red;
                    }
