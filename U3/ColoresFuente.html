Introducción

El CSS proporciona varias propiedades para aplicar colores y fondos a los elementos HTML. Estas propiedades son esenciales para mejorar el aspecto visual y la legibilidad de la página web.

Colores y Fondos

  1. Propiedad color

    La propiedad color se utiliza para cambiar el color del texto dentro de un elemento. Puedes definir los colores de varias maneras:

    1. Nombres de colores: utilizar nombres predefinidos cómo red, blue, green, etc.
    2. Colors hexadecimals: utilitzar un codi hexadecimal que comença amb # seguit de sis caràcters (exemple: #FF0000 per a vermell).
    3. Colores hexadecimales: utilizar un código hexadecimal que empieza con # seguido de seis caracteres (ejemplo: #FF0000 para rojo).
    4. RGB: utilizar valores de rgb() donde cada componente (rojo, verde, azul) se especifica entre 0 y 255.
    5. RGBA: igual que rgb(), pero con un cuarto valor para la opacidad (valores entre 0.0 y 1.0). 
                        p {
                        color: blue; 
                    }
                    h1 {
                        color: #FF5733; 
                    }
                    span {
                        color: rgb(0, 255, 0); 
                    }
                    div {
                        color: rgba(0, 0, 255, 0.5); 
                        
                    }
  2. Propiedad de Background color

    La propiedad background-color se utiliza para establecer el color de fondo de un elemento. Igual que la propiedad color, puedes definir colores utilizando nombres, valores hexadecimales, RGB o RGBA. 

                        body {
                        background-color: lightgray; /* Color de fons gris clar */
                    }
                    div {
                        background-color: #00FF00; /* Color de fons verd */
                    }
  3. Propiedad Background

    La propiedad background es una manera abreviada de especificar varias propiedades de fondos a la vez, como el color, la imagen de fondo, la repetición, la posición, etc. 

                        div {
                        background: lightblue url("fons.jpg") no-repeat center cover;
                    }
  4. Propiedad background-image

    Esta propiedad permite añadir una imagen como fondo de un elemento. Puedes especificar la ubicación de la imagen con una URL. 

                        div {
                        background-image: url("imatge-de-fons.jpg");
                    }

    Si quieres que la imagen se repita o se muestre de otro modo, puedes utilizar otras propiedades complementarias:

    1. background-repeat: Controla si la imagen de fondo se repite. Los valores pueden ser repeat, no-repeat, repeat-x (horizontalmente) o repeat-*y (verticalmente).
    2. background-position: Define la posición de la imagen de fondo (por ejemplo, center, top, left).
    3. background-size: Permite ajustar la medida de la imagen (por ejemplo , cover o contain para adaptarse a la medida del elemento). 
                        div {
                        background-image: url("fons.jpg");
                        background-repeat: no-repeat;
                        background-position: center;
                        background-size: cover;
                    }

Unidades Comunes

  1. Píxeles (px): Unidad fija, representa un punto a la pantalla. Ejemplo: margin: 20px;
  2. Ems (me): Unidad relativa a la medida de la fuente del elemento pare. Ejemplo: padding: 2em; (2 veces la medida de la fuente).
  3. Remos (remo): Unidad relativa a la medida de la fuente del elemento raíz (HTML). Ejemplo: margin: 1.5rem; (1.5 veces la medida de la fuente raíz).
  4. Porcentaje (%): Relativa a la anchura o altura del contenedor pare. Ejemplo: padding: 5%; (5% de la anchura del contenedor).
  5. Viewports (vw, vh): Relativa a la medida de la ventana visible. Ejemplo: margin: 5vw; (5% de la anchura del viewport).
  6. Bordes (border-*width): Grosores predeterminados: thin, medium, thick
