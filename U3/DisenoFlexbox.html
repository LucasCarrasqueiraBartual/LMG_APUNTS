Introducción

Flexbox es un sistema de diseño en CSS que proporciona una manera muy eficiente de distribuir el espacio entre los elementos y alinearlos dentro de un contenedor, incluso cuando la medida de los elementos es desconocida o dinámica. Flexbox es ideal para crear diseños de páginas adaptativas, puesto que facilita la alineación, distribución y ordenación de los elementos dentro de un contenedor.

Diseño Basico Flexbox

  1. Contenedor flex

    El punto de partida para utilizar Flexbox es establecer display: flex en el elemento contenedor. Este contenedor se conoce como contenedor flex, y sus elementos hijos se conocen como elementos flex. Una vez activada el display: flex, puedes utilizar varias propiedades para ajustar como se distribuyen los elementos dentro del contenedor. 

                        .contingut {
                        display: flex;
                    }
  2. Direccion del flexbox

    Por defecto, los elementos flex se disponen en fila horizontalmente. Puedes cambiar la dirección con la propiedad flex-*direction.

    1. fila (row): Los elementos se colocan en fila (horizontal) (valor por defecto).
    2. fila inversa (row-reverse): Los elementos se colocan en fila pero en orden inverso.
    3. columna (column): Los elementos se colocan en columna (vertical).
    4. columna inversa (column-reverse): Los elementos se colocan en columna pero en orden inverso.
                        .contenidor {
                        display: flex;
                        flex-direction: column;
                    }
  3. Alineamiento horizontal

    La propiedad justify-contento controla la alineación de los elementos a lo largo del eje principal (horizontal por defecto). Puedes distribuir los elementos flex de diferentes maneras dentro del contenedor. 

                        .contenidor {
                        display: flex;
                        justify-content: center;
                    }
  4. Alineamiento vertical

    La propiedad align-*items controla la alineación de los elementos a lo largo del eje transversal (vertical si la dirección es horizontal). Esta propiedad determina como se alinean los elementos flex dentro del contenedor en relación con su altura. 

                        .contenidor {
                        display: flex;
                        align-items: center;
                    }
  5. Control del espacio que ocupa cada elemento

    La propiedad flex-*grow especifica como los elementos flex crecen dentro del contenedor para ocupar el espacio disponible. Cada elemento se puede asignar un valor numérico que indica cuánto tiene que crecer en relación con los otros elementos. Si todos los elementos tienen flex-*grow: 1, el espacio disponible se distribuirá de manera uniforme. 

                        .element {
                        flex-grow: 2; 
                    }
