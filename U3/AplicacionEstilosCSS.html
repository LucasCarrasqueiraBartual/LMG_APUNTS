Aplicacion de Estilos

El CSS se puede aplicar a un documento HTML de tres maneras principales: en línea, incrustado (dentro del HTML) y externo (mediante un fichero CSS separado). Cada una de estas formas tiene sus propósitos y usos..

Diferentes Estilos

  1. Estilos en línea (Inline styles)

    2. Los estilos en línea se aplican directamente a un elemento HTML, utilizando el atributo style dentro de la misma etiqueta del elemento. Este método es adecuado para aplicar estilos a un solo elemento de manera rápida, pero no es recomendable para páginas grandes, puesto que dificulta la mantenibilitat.

    <p style="color: red; font-size: 20px;">Esto es un parrafo.</p> Resultado:

    Esto es un parrafo.

  2. Estilos incrustados (Embedded styles)

    3. Los estilos incrustados se colocan dentro de la sección <head> de un documento HTML, dentro de una etiqueta <style>. 

    <!DOCTYPE html>
<html lang="ca">
  <head>
    <meta charset="UTF-8">
    <title>Estils incrustats</title>
    <style>
      p {
        color: blue;
        font-size: 18px;
      }
      h1 {
        text-align: center;
      }
    </style>
  </head>
  <body>
    <h1>Aquest és un títol centrat</h1>
    <p>Aquest paràgraf tindrà text de color blau i mida 18 píxels.</p>
  </body>
</html>
  3. Estilos externos (External stylesheets)

    4. Los estilos externos se almacenan en un fichero CSS separado, y se enlazan al documento HTML utilizando la etiqueta <link> dentro de la sección <head>. Este es el método más recomendado para la mayoría de lugares webs, especialmente si constan de múltiples páginas.

El Archivo HTML: 

          <!DOCTYPE html>
          <html lang="ca">
            <head>
              <meta charset="UTF-8">
              <title>Estils externs</title>
              <link rel="stylesheet" href="style.css">
            </head>
            <body>
              <h1>Benvinguts al meu lloc web</h1>
              <p>Aquest paràgraf tindrà text verd i mida 16 píxels.</p>
            </body>
          </html>

El Archivo CSS:

            p {
              color: green;
              font-size: 16px;
              }
            h1 {
              text-align: center;
              color: darkblue;
              }
