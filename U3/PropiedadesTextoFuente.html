Propiedades Texto y Fuente

El CSS permite controlar como se muestran los textos dentro de una página web. Con varias propiedades, podemos cambiar la fuente, el color, la alineación, la medida del texto, y otros aspectos visuales para mejorar la legibilidad y la estética del contenido.

  1. Propiedad font-family:

    La propietat font-family defineix el tipus de lletra (tipografia) que s'utilitzarà per a un element de text. Pots especificar una o més fonts, i el navegador aplicarà la primera que estigui disponible al sistema de l'usuari. 

                        p {
                        font-family: "Arial", sans-serif;
                    }
  2. Propiedad font-size:

    La propietat font-size controla la mida de la lletra. Els valors es poden definir en diverses unitats, com píxels (px), ems (em), punts (pt), o percentatges (%). 

     
                    h1 {
                        font-size: 32px;
                    }

                    p {
                        font-size: 14px;
                    }
  3. Propiedad color

    La propiedad color define el color del texto. Se pueden utilizar nombres de colores predefinidos, valores en hexadecimal, o el sistema de colores RGB o RGBA. 

                        p {
                        color: red; 
                    }
                    h1 {
                        color: #00FF00; 
                    }
                    span {
                        color: rgb(0, 0, 255); 
                    }
  4. Propiedad text-align

    La propiedad texto-align se utiliza para controlar la alineación del texto dentro de un elemento. Los valores más comunes son left (izquierda), right (derecha), center (centrado) y justify (justificado). 

                        h1 {
                        text-align: center;
                    }
                    p {
                        text-align: justify;
                    }
  5. Propiedad font-weight

    La propiedad fuente-weight controla el grosor de la letra. Los valores comunes son:

    • normal: por defecto.
    • bold: texto en negrita.
    • Números entre 100 y 900, donde 400 es el valor por defecto y 700 representa negrita. 

                        p {
                        font-weight: bold;
                    }
  6. Propiedad font-style

    La propiedad fuente-style controla el estilo de la fuente, como si el texto será normal o en cursiva.
    Els valors més comuns són:

    • normal: por defecto.
    • italic: texto en cursiva.
    • oblique: texto inclinado. 

                        span {
                        font-style: italic;
                    }
